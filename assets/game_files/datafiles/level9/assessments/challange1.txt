Q: Which keyword is used to catch exceptions in Python?
O: catch
   except
   finally
   raise
A: except

Q: What will happen if an exception occurs inside a try block but there is no except block?
O: The program continues
   The program crashes
   The error is ignored
   The program executes the next line
A: The program crashes

Q: The finally block always executes, __________ of exceptions.
O: when an error occurs
   before exceptions
   regardless
   only in case of an error
A: regardless

Q: The raise keyword is used to __________ an exception manually.
O: ignore
   catch
   throw
   handle
A: throw

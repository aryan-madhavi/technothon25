Q: What does the break statement do?
O: Skips the next iteration
   Stops the loop completely
   Restarts the loop
   Does nothing
A: Stops the loop completely

Q: What does the continue statement do?
O: Stops the loop
   Skips the current iteration
   Restarts the loop
   None
A: Skips the current iteration

Q: The break statement is used to __________ a loop prematurely.
O: Exit
   Skip
   Restart
   Iterate
A: Exit

Q: The continue statement skips the __________ of a loop.
O: Current iteration
   Next iteration
   Entire loop
   None
A: Current iteration

Q: The continue statement stops the loop.
O: True
   False
A: False
